# geoAPI
RestAPI that provide ip Geolocalisation base on GeoLite2 data. 

# Requirement 

Tested on ubuntu 16.04 LTS 

	python3
	apt-get install python3-pip
	pip3 install flask
	pip3 install flask-restful 

# Running 

	python3 geoApi.py
